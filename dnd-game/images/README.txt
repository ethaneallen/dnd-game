D&D Game - Campaign Cover Images
==================================

Place your campaign book cover images in this folder with the following names:

1. nights_dark_terror.jpg - Cover for Night's Dark Terror (B10 module)
2. curse_of_strahd.jpg - Cover for Curse of Strahd

Recommended image size: 300x400 pixels (or similar 3:4 aspect ratio)
Supported formats: .jpg, .png, .webp

The game will automatically use these images if they exist.
If an image is not found, a styled fallback will be displayed.
