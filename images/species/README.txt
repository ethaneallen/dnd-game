================================================================================
                    CHARACTER SPECIES PORTRAIT IMAGES
================================================================================

Place one portrait image per race/species in this folder.
The game will automatically detect and display them on the character panel.

REQUIRED FILENAMES (case-sensitive, .jpg preferred, .png also supported):
--------------------------------------------------------------------------------

  human.jpg         - Human character portrait
  elf.jpg           - Elf character portrait
  dwarf.jpg         - Dwarf character portrait
  halfling.jpg      - Halfling character portrait
  dragonborn.jpg    - Dragonborn character portrait
  tiefling.jpg      - Tiefling character portrait
  half-orc.jpg      - Half-Orc character portrait

RECOMMENDED SPECS:
--------------------------------------------------------------------------------

  - Size: 128x128 px to 256x256 px (square aspect ratio)
  - Format: PNG (transparent background preferred) or JPG
  - Style: Head/bust portrait, fantasy art style
  - The image will be displayed in a 48px circular frame with gold border

SOURCING IMAGES:
--------------------------------------------------------------------------------

  You can find free-to-use D&D-style character portraits from:

  1. Hero Forge Screenshots (heroforge.com)
     - Create a free character model, screenshot the bust

  2. Open Game Art (opengameart.org)
     - Search "fantasy portrait" — many CC0/CC-BY licensed assets

  3. Dungeon Master's Guild (dmsguild.com)
     - Some free community art packs with character portraits

  4. Stable Diffusion / AI Art (locally generated)
     - Generate your own fantasy portraits for personal use

  5. D&D Beyond Free Assets
     - Some basic race illustrations are available

  DO NOT use official Wizards of the Coast artwork without a license.

FALLBACK:
--------------------------------------------------------------------------------

  If an image file is missing for a race, the game will show the emoji
  fallback instead (e.g., 🧑 for Human, 🧝 for Elf, etc.).

================================================================================
